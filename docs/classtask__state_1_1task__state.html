Detailed Description

Discrete-time Luenberger state observer for the Romi robot.

State vector: x = [s (mm), psi (rad), omL (rad/s), omR (rad/s)]
Update law:   x_{k+1} = AD @ x_k + BD @ u*_k
where u* = [uL, uR, sL, sR, psi, psidot]
